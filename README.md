# Filegap Filegap

## How do I install these formulae?

`brew install filegap/filegap/<formula>`

Or `brew tap filegap/filegap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "filegap/filegap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
